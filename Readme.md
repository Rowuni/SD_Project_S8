## Launch the project

- Clone the repository
- Run the file run_demo.ps1 to test all the features

If the .ps1 file have an error while executing, it's because of your execution policy of powershell files.
Check this link for help: https://learn.microsoft.com/en-us/powershell/module/microsoft.powershell.security/set-executionpolicy?view=powershell-7.6
